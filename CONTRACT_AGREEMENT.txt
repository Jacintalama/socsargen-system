                           ARTICLE I - PROJECT SCOPE
--------------------------------------------------------------------------------

1.1 PROJECT DESCRIPTION

The DEVELOPER agrees to design, develop, and deliver a Hospital Management
System ("System") for the CLIENT with the following features:

A. PUBLIC WEBSITE FEATURES:
   - Responsive homepage with hospital information
   - Services and departments listing
   - Doctor profiles and schedules
   - Contact information and location
   - News and announcements section
   - AI-powered chatbot assistant (Gemini Flash)

B. PATIENT PORTAL:
   - User registration and authentication
   - Online appointment booking system
   - Appointment history and management
   - Profile management
   - Real-time chat support

C. DOCTOR DASHBOARD:
   - Appointment management
   - Patient records access
   - Schedule management
   - Profile settings

D. ADMIN DASHBOARD:
   - User management (patients, doctors, staff)
   - Appointment oversight
   - Service and department management
   - News/announcements management
   - Analytics and reports
   - System settings

1.2 TECHNICAL SPECIFICATIONS

The System shall be built using the following technologies:

Frontend:
   - React.js (JavaScript Framework)
   - Tailwind CSS v4 (Styling)
   - React Router (Navigation)
   - Socket.io Client (Real-time features)

Backend:
   - Node.js with Express.js
   - PostgreSQL Database
   - JWT Authentication
   - Socket.io (Real-time communication)
   - Google Gemini Flash API (AI Chatbot)

Hosting Requirements (to be provided by CLIENT):
   - Web hosting with Node.js support
   - PostgreSQL database server
   - SSL Certificate
   - Domain name
